# PromptVault

> **Git for your prompts.** Version control, diff, branch, and A/B test LLM prompts across OpenAI, Anthropic, and Gemini.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Stack](https://img.shields.io/badge/stack-Next.js%20%7C%20NestJS%20%7C%20Supabase-blue)](#)

## Problem
Teams write hundreds of prompts with zero version control. A prompt that worked last week breaks today — no history, no diff, no rollback.

## Solution
PromptVault treats prompts as first-class versioned artifacts — just like source code.

## Key Features
- 📝 Rich prompt editor with variable support
- 🔀 Version history with commit messages and diff viewer
- 🧪 Multi-provider test runner (OpenAI · Anthropic · Gemini)
- 👥 Team workspaces with roles
- 📊 A/B testing and performance analytics
- 🔌 API + CLI for CI/CD integration

## Tech Stack
`Next.js 14` · `TypeScript` · `NestJS` · `Supabase` · `Tailwind CSS`
